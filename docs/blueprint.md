# **App Name**: UserHub

## Core Features:

- Account Creation: User account creation with email verification.
- User Login: Secure user login with session management.
- Personalized Dashboard: Display of information specific to the logged-in user.
- Route Protection: Authentication and authorization middleware to protect user-specific routes.
- Profile Settings: Profile settings page where users can update their personal information.

## Style Guidelines:

- Primary color: Subtle blue (#6699CC), evoking trust and security.
- Background color: Light gray (#E0E0E0), offering a clean and modern feel.
- Accent color: Warm orange (#FF8C00), used for calls to action.
- Body and headline font: 'Inter', a versatile sans-serif providing a clean and accessible reading experience.
- Use clear and simple icons to represent user data and settings.
- A clean, card-based layout to display information in a modular and digestible format.
- Subtle transitions and animations to enhance user experience, such as fading in user data upon login.
